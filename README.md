# ConWeb - Gestión de Contactos

Proyecto final DAW desarrollado con HTML, CSS, JavaScript, PHP y MySQL.

ConWeb es una aplicación web para gestionar contactos personales y profesionales mediante categorías y un sistema de usuarios.

## Funcionalidades

- Registro de usuarios
- Inicio y cierre de sesión
- Crear, editar y eliminar contactos
- Crear y editar categorías
- Asignar categorías a contactos
- Exportar contactos a CSV
- Editar perfil de usuario
- Cambio de contraseña
- Últimos accesos
- Panel principal con estadísticas
- Diseño responsive para móvil

## INSTALACIÓN

1. Copiar la carpeta del proyecto dentro de:

    /Applications/MAMP/htdocs

2. Abrir phpMyAdmin y crear una base de datos llamada:

    conweb

3. Importar el archivo:

    /database/conweb.sql

4. Iniciar Apache y MySQL desde MAMP.

5. Abrir en el navegador:

    http://localhost:8888/ConWeb/pages/login.php
